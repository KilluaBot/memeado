# Drake Like Hate

Generates PetPet meme

**URL** : `/api/drakelikehate`

**Method** : `GET`

**Auth required** : NO

**Data constraints**

```json
{
    "text1": "[Plain text]",
    "text2": "[Plain text]"
}
```

**Data example**

```json
{
    "text1": "Example text 1",
    "text2": "Example text 2"
}
```

## Success Response

**Code** : `200 OK`
